Plus Twitter Slider is Responsive Twitter Slider Sidebar Plugin.

== Description ==

Download Twitter Slider to connect your website visitors with your Twitter account. This multi Twitter plugin easily displays the latest tweets and Twitter activity. Even better, Twitter Slider comes with a simple installation process. After downloading our Twitter sidebar, just add your Twitter user account name for instant access to your Twitter account. Once the Twitter Slider installation process is completed, a Twitter sidebar will be placed on your web page, displaying up to the minute Twitter activity including tweets and Twitter news updates. Choose Twitter Slider today for all of your website, Twitter, and social media needs.

You can look at demo here: http://twitterslider.com/wp/

== Installation ==

Thanks for installing or downloading our Twitter Slider Plugin

1. Install the wordpress plugin first
You can install it via Wordpress Installer or can download the file first then Upload it on 'wp-content' folder

2. Activate the plugin

3. You will see facebook settings from right side of your wordpress menu.
